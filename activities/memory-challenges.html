Memory Strategy

Simple memory routine for children

Use this method before every challenge to make memory practice calm, fun, and effective.

1. Look

Read the list, numbers, or instruction slowly.

2. Picture

Make a picture in your mind.

3. Repeat

Say it softly once or twice.

4. Recall

Close your eyes, remember, then answer.
