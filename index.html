Parent trust framework

Clear policies before big promises.

This draft avoids invented testimonials, certifications, and achievements. Trust should be earned through transparent teacher profiles, child safety rules, parent communication, visible lesson structure, and honest outcomes.

Ask about teacher matching

Child safety statement

Children learn in a supervised, respectful environment with parent communication and clear class expectations.

Teacher quality statement

Teacher profiles should include verified experience, teaching style, age suitability, and subject strengths.

Progress tracking statement

Parents receive simple updates showing attendance, lesson focus, practice goals, and progress milestones.

FAQs

Questions parents usually ask first.

Is this suitable for complete beginners?

Yes. The first step is a level check so the child can start from an appropriate point.

Do you offer online classes?

The site is prepared for online one-to-one and group class inquiries. Confirm exact availability before launch.

How do parents track progress?

Progress updates should include attendance, lesson focus, practice notes, and next milestones.

Are teacher profiles verified?

Teacher details should be added only after verification. Do not publish unconfirmed credentials.
