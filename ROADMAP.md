# Roadmap

## Completed In This Version

- Converted the site from a marketing-style platform into a mobile-first children's learning app.
- Reworked the homepage around ages 3-8 and the flow: Age -> Daily Path -> Lesson -> Practice -> Quiz -> Reward -> Parent Follow-up.
- Added subject pages for English, Mathematics, Science, Arabic, Islamic Studies, General Knowledge, and Computer Skills.
- Added quiz, worksheets, activities, practice, reading library, Islamic corner, parent corner, privacy, 404, sitemap, robots, and web manifest.
- Preserved older public URLs and redirected their content toward the new learning structure.

## Next Improvements

- Add more story pages with illustrations and source-reviewed Islamic stories.
- Add a larger static quiz bank by age and subject.
- Add more printable worksheets.
- Add verified teacher profiles only after credentials are confirmed.
- Submit `sitemap.xml` in Google Search Console after upload.

## Quality Rules

- Ages 3-8 remains the consistent target.
- No child login.
- No sensitive child data.
- No fabricated religious claims or testimonials.
- Static HTML content must remain visible even if JavaScript fails.
