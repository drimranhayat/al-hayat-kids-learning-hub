# Al-Hayat Kids Learning Hub

Static GitHub Pages website for Islamic and academic learning for children ages 3-8.

## What This Version Includes

- Mobile-first child learning gateway
- Consistent target age range: ages 3-8
- Age groups: ages 3-4, 5-6, and 7-8
- Daily learning path: word, number, story, dua/manner, mini quiz
- Subject mini-courses for English, Mathematics, Science, Arabic, Islamic Studies, General Knowledge, and Computer Skills
- Quiz page with static fallback questions and JavaScript enhancement
- Worksheets with printable sections
- Activities and practice pages
- Reading library with story questions, moral lesson, and parent follow-up
- Islamic corner with authentic common duas and shared Islamic values
- Parent corner with local progress, safety badges, and printable certificate
- Privacy page, 404 page, sitemap, robots, and web manifest

## Privacy and Safety

- No child login required
- No sensitive child data collection
- Progress is stored locally in the browser only
- Parent supervision is recommended for ages 3-8
- Islamic content avoids fabricated hadith, weak claims, and sectarian tone
