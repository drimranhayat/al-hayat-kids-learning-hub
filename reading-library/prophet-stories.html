Authenticity Note

Careful source-aware summaries

This page gives age-appropriate summaries and references. It avoids long translations and avoids adding details where the Qur’an gives only brief mention.

Primary Qur’an reference style

Each main prophet card includes Qur’an references for parents and teachers. Children should learn prophet stories through careful explanation, not unsupported legends.

Important classroom rule

  • Do not invent details for briefly mentioned prophets.
  • Use child-friendly language for serious stories.
  • Keep Muhammad ﷺ’s full Seerah on the dedicated Seerah page.
  • Use stories to build worship, manners, and character.
